• 在命令行执行命令，npm install -g nrm，全局安装nrm。执行命令nrm ls查看可选的源。如果要切换到taobao源，执行命令nrm use taobao。

  • 通过-v测试版本号
  • 先在根目录创建src文件夹里面有index.html和main.js
  • 创建dist文件夹，package.json文件和webpack.config.js文件
  • 在cmd中输入npm install --save-dev webpack，npm install --save-dev webpack-cli
  • vuecli后面已经没有webpack.config.js，移到了build的webpack.base.conf.js，
  • 注：如果在Terminal中直接运行webpack命令不成功，使用node_modules/.bin/webpack执行

  • 使用 webpack-dev-server 这个工具，来实现自动打包编译的功能

  • 运行 npm i webpack-dev-server -D
  • 在package.json中的scripts配置

    "test": "echo \"Error: no test specified\" && exit 1"

    "dev": "webpack-dev-server --open --port 3000 --contentBase src --hot"

  • 注：webpack-dev-server 这个工具，如果想要正常运行，要求，在本地项目中，必须安装 webpack

  • 辅助包的引入

  • npm install jquery --save-dev 安装jquery
  • npm install bootstrap --save-dev 安装bootstrap
  • 在main.js文件中import $ from 'jquery'和import 'bootstrap/dist/css/bootstrap.css'
  • 注：看在package.json中有无此配置

    "dependencies": {
    "bootstrap": "^3.3.7",
    "vue": "^2.5.2"
    }

  • 使用webpack打包css文件

  • npm i style-loader css-loader --save-dev
  • 修改`webpack.config.js`这个配置文件：

    module: {
    rules: [
    { test: /\.css$/, use: ['style-loader', 'css-loader'] }
    ]
    }

  • 使用webpack打包less sass文件

  • npm i less-loader less -D cnpm i sass-loader node-sass --save-dev
  • 修改`webpack.config.js`这个配置文件:

    { test: /\.less$/, use: ['style-loader', 'css-loader', 'less-loader'] },
    { test: /\.scss$/, use: ['style-loader', 'css-loader', 'sass-loader'] },

  • 使用webpack处理css中的路径

  • npm i url-loader file-loader --save-dev
  • 修改`webpack.config.js`这个配置文件:

    { test: /\.(png|jpg|gif)$/, use: 'url-loader' }
    可以通过`limit`指定进行base64编码的图片大小；只有小于指定字节（byte）的图片才会进行base64编码：
    { test: /\.(png|jpg|gif)$/, use: 'url-loader?limit=43960' },

  • 使用babel处理高级JS语法

  • npm i babel-core babel-loader babel-plugin-transform-runtime --save-dev
  • npm i babel-preset-env babel-preset-stage-0 --save-dev
  • 修改`webpack.config.js`这个配置文件:

    { test: /\.js$/, use: 'babel-loader', exclude: /node_modules/ }

  • 在项目根目录中添加`.babelrc`文件，并修改这个配置文件如下：

    {
    "presets":["es2015", "stage-0"],
    "plugins":["transform-runtime"]
    }
